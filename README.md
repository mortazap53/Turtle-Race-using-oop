# Turtle-Race-using-oop

This project is a <<<multi‑player Turtle Racing betting>>> game built with Python’s turtle graphics library. Players place bets on turtle colors, turtles race across the screen with random movement, and the game announces winners based on finishing positions.

And the objective of making this project is, having a better understanding of python logic and poop python object-oriented programming.
